## Dependencias

Esto utiliza `sox`.

### Arch Linux

```bash
sudo pacman -S sox
```

### Debian / Ubuntu

```bash
sudo apt install sox
```

## Instalación

Clona el repositorio:

```bash
git clone URL_DEL_REPOSITORIO
```

Entra a la carpeta:

```bash
cd nombre
```

Da permisos al instalador:

```bash
chmod +x install.sh
```

Ejecuta el instalador:

```bash
sudo ./install.sh
```

## Uso

Después de instalarlo simplemente ejecuta:

```bash
67
```
